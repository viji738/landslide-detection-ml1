# 🌍 Landslide Detection Model using Machine Learning

## 📌 Internship Project – NovaNectar Services Pvt. Ltd.

This project was developed as part of my internship, focusing on predicting landslide-prone areas using historical data and Machine Learning models.

---

## 🔍 Project Objective

To build a predictive classification model capable of identifying landslide-prone regions using 20 years of historical dataset.

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Random Forest
- Support Vector Machine (SVM)
- Convolutional Neural Network (CNN)
- Kaggle Dataset

---

## ⚙️ Project Workflow

1. Data Collection from Kaggle
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Model Training using:
   - Random Forest
   - Support Vector Machine (SVM)
5. Model Evaluation & Accuracy Analysis
6. Exploratory CNN implementation for image-based classification

---

## 📊 Key Contributions

- Processed and analyzed large real-world datasets
- Implemented and compared multiple ML algorithms
- Performed hyperparameter tuning
- Evaluated model performance to improve prediction accuracy
- Explored CNN for image-based learning

---

## 🎯 Learning Outcomes

- Model comparison and evaluation techniques
- Real-world data handling and preprocessing
